# ⚽ FootballPro - Java 22/23 Console-Based Football Player Management System

### 👨‍💻 Developed by [SanketJr11](https://github.com/SanketJr11)

---

## 📝 Project Overview
**FootballPro** is a console-based football player management system built using **modern Java 22/23 features**.  
It demonstrates both **fundamental OOP principles** and **advanced Java concepts**, making it a perfect academic and practical project example.

---

## 🚀 Features Demonstrated

### 🧩 Fundamentals
- ✅ Classes, Objects, and Encapsulation  
- ✅ Method Overloading and Varargs  
- ✅ `this()` vs `this.` usage  
- ✅ Inheritance, Overriding, and Polymorphism  
- ✅ `super()` and `super.` contrasts  
- ✅ Interfaces and Implementation  
- ✅ Enums and Arrays / ArrayList  
- ✅ Checked & Unchecked Exceptions  
- ✅ Java Core API: `StringBuilder`, `LocalDate`

### ⚙️ Advanced Java 21–23
- ✅ Sealed Classes and Interfaces  
- ✅ Records  
- ✅ Lambdas and Predicates  
- ✅ Method References  
- ✅ Defensive Copying  
- ✅ Switch Expressions & Pattern Matching  
- ✅ Immutable Custom Type  
- ✅ Local Variable Type Inference (`var`)  
- ✅ Interface Private, Default, and Static Methods

## 🧠 Application Domain
**Domain:** Sports (Football)  
**Goal:** Manage player details — add, view, search, and display match statistics.

## 🧩 How to Run (macOS / Linux / Windows)

### 1️⃣ Compile
```bash
javac com/football/Main.java com/football/model/*.java com/football/service/*.java com/football/record/*.java com/football/exception/*.java com/football/util/*.java
```

### 2️⃣ Run
```bash
java com.football.Main
```
