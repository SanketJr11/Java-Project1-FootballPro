Football Player Management System (Console, Java 22/23)
======================================================

macOS QUICK START
-----------------
1) Open Terminal and cd into this folder:
   cd FootballPro

2) Compile (zsh, safe explicit form):
   javac com/football/Main.java com/football/model/*.java com/football/service/*.java com/football/record/*.java com/football/exception/*.java com/football/util/*.java

   (Alternative using recursive globs if enabled)
   javac com/football/**/*.java com/football/*.java

3) Run:
   java com.football.Main

WHAT THIS DEMO SHOWS
--------------------
- Sealed classes, records, switch expressions, LVTI (var)
- Inheritance, polymorphism, interfaces with default/static/private methods
- Enums, exceptions, varargs, method overloading, StringBuilder, Date API
- Lambdas, method references, defensive copying
